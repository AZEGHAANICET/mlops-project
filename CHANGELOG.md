# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Iris Classifier MLOps pipeline
- FastAPI application with web interface and REST API
- Jinja2 templates with Tailwind CSS for modern UI
- Machine learning model training script
- DVC integration for data and model versioning
- Docker support with Dockerfile and docker-compose.yml
- Comprehensive test suite with pytest
- Health check and metrics endpoints
- Input validation with Pydantic
- Error handling and logging
- Professional README with Mermaid diagrams
- Documentation and contribution guidelines

### Features
- Real-time iris flower classification
- Web interface for predictions
- REST API for programmatic access
- Model metrics tracking
- Containerized deployment
- Data versioning with DVC
